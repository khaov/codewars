(7 kyu) Make a square box!

Easy; Make a box

Given a number as a parameter (between 2 and 30), return an array containing strings which form a box.

Like this:

n = 5

  
    [
      '-----',
      '-   -',
      '-   -',
      '-   -',
      '-----'
    ]

n = 3

  
    [
      '---',
      '- -',
      '---'
    ]
